Icon button style is applied if a button contains no text. This works for buttons of type `sbb-button`, `sbb-alt-button`, `sbb-secondary-button` and `sbb-ghost-button`.

sbb-button

sbb-alt-button (lean only)

This button is only intended to be used with lean variant.

sbb-ghost-button

In standard variant, this button will be hidden when disabled.
