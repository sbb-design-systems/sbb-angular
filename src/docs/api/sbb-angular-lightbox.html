API reference for Angular SACL lightbox

import { LightboxModule } from 'sbb-angular';

Services

Lightbox

Service to open SBB Design modal lightboxes.

Properties
Name Description

afterAllClosed: Observable<void>

Stream that emits when all open lightbox have finished closing. Will emit on subscribe if there are no open lightboxes to begin with.

afterOpen: Subject<LightboxRef<any>>

Stream that emits when a lightbox has been opened.

openLightboxes: LightboxRef<any>[]

Keeps track of the currently-open lightboxes.
Methods
closeAll

Closes all of the currently-open lightboxes.
getLightboxById

Finds an open lightbox by its id.
Parameters

id

string

ID to use when looking up the lightbox.
Returns
LightboxRef<any> | undefined

Lightbox reference associated to the input id.
open

Opens a modal lightbox containing the given component.
Parameters

componentOrTemplateRef

ComponentType<T> | TemplateRef<T>

Type of the component to load into the lightbox, or a TemplateRef to instantiate as the lightbox content.

config?

LightboxConfig<D>

Extra configuration options.
Returns
LightboxRef<T, R>

Reference to the newly-opened lightbox.

Additional classes

LightboxConfig

Configuration for opening a modal lightbox with the Lightbox service.

Properties
Name Description

ariaDescribedBy: string | null

ID of the element that describes the Lightbox.

ariaLabel: string | null

Aria label to assign to the Lightbox element

autoFocus: true

Whether the Lightbox should focus the first focusable element on open.

closeOnNavigation: true

Whether the Lightbox should close when the user goes backwards/forwards in history.

data: D | null

Data being injected into the child component.

disableClose: false

Whether the user can use escape or clicking on the backdrop to close the modal.

height: '100vh'

Height of the Lightbox.

id: string

ID for the Lightbox. If omitted, a unique one will be generated.

panelClass: string | string[]

Custom class for the overlay pane.

role: LightboxRole

The ARIA role of the Lightbox element.

scrollStrategy: ScrollStrategy

Scroll strategy to be used for the dialog.

viewContainerRef: ViewContainerRef

Where the attached component should live in Angular's logical component tree. This affects what is available for injection and the change detection order for the component instantiated inside of the Lightbox. This does not affect where the Lightbox content will be rendered.

width: '100vw'

Width of the Lightbox.
