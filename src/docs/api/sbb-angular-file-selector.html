Additional interfaces

FileSelectorOptions

Properties
Name Description

accept: string

Categories of file types accepted by sbb-file-selector component. It is optional.

capture: 'user' | 'environment'

Attribute whose state specifies the preferred facing mode for the media capture mechanism. The attribute's keywords are user and environment, which map to the respective states user and environment. It is optional.

multiple: boolean

Mode on file selector component to chose more files to upload. It is optional.

multipleMode: 'default' | 'persistent'

Set if the component should add files on top of the already selected ones or keep default input file behaviour
