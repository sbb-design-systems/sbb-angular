Additional classes

FormFieldControl

Properties
Name Description

autofilled: boolean

Whether the input is currently in an autofilled state. If property is not present on the control it is assumed to be false.

disabled: boolean

Whether the control is disabled.

empty: boolean

Whether the control is empty.

errorState: boolean

Whether the control is in an error state.

focused: boolean

Whether the control is focused.

id: string

The id of the form field.

inputId: string

The id of the inner input field. Can be the same as the id property.

ngControl: NgControl | undefined

The attached NgControl, if any exists.

required: boolean

Whether the control is required.

stateChanges: Observable<void>

Stream that emits whenever the state of the control changes such that the parent MatFormField needs to run change detection.

value: TValue | null

The value of the control.
