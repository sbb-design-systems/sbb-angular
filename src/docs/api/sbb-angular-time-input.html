API reference for Angular SACL time-input

import { TimeInputModule } from 'sbb-angular';

Directives

TimeInputDirective

Selector: input[sbbTimeInput]

Properties
Name Description

inputmode: 'numeric'

Input mode for time input

pattern: '[0-9]*'

Allowed pattern for time input value

ssbTimeInput: true

Class property that refers to time input

type: 'text'

Value type allowed
Methods
onBlur

Method that sets the input value in 'hours:mins' format on blur event
